IP CIDR → VLAN Planner

Enter a parent CIDR and load a vlan-structure.txt file to allocate subnets per VLAN.

📄 Click or drop vlan-structure.txt
vlan-structure.txt format  —  One VLAN per line, three comma-separated values: vlan_id, description, prefix_length. Blank lines are ignored. The prefix can be written as 24, /24, or dotted-decimal 255.255.255.0. To silently reserve a subnet without showing it in the output, set the description to RESERVED — the address space will still be consumed.
Examples:  10,SERVER_VLAN,24  |  220,DEVICE_MANAGEMENT,/24  |  240,WAN_TRANSIT,255.255.255.240  |  300,RESERVED,24
